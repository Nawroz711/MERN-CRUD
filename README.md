# MERN CRUD Application

This is a full-stack MERN (MongoDB, Express.js, React.js, Node.js) application that provides CRUD (Create, Read, Update, Delete) operations.

## Features

- User authentication (login/signup)
- Create, read, update, and delete posts
- Responsive frontend built with React and Vite
- Backend API with Express.js
- MongoDB for data storage

## Getting Started

### Prerequisites

- Node.js
- MongoDB
- npm or yarn

### Installation

1. Clone the repository
2. Install backend dependencies: `cd backend && npm install`
3. Install frontend dependencies: `cd frontend && npm install`
4. Set up environment variables in backend/.env
5. Start the backend: `cd backend && npm start`
6. Start the frontend: `cd frontend && npm run dev`

## Usage

- Register a new user or login
- Create, view, edit, and delete posts
